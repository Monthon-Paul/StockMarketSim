Top Section:

New Portfolio:

Creates a new Portfolio for the User. If there is Unsave changes, it will ask the User (you) want to save. "Yes" will ask you to name it, Saving the file will create a new file. Otherwise if it does exist, it updates the pre-existing file. "No" will creates new Portfolio without saving.

Load Portfolio:

Loads in an already saved Portfolio for the User. To open a file, it should end in ".stk" for this Program. If the Portfolio has unsave changes, it will ask if you want to save. "Yes" will ask you to name it, Saving the file will create a new file. Otherwise if it already exist, it updates the pre-existing file. "No" will discard it changes. Otherwise, Loads in a Portfolio for the User.

Quick Save:

Saves the Portfolio. If it already exist, quickly save the contents & update the file. Otherwise, same feature as Save Portfolio.

Save Portfolio:

Let the User (you) save the Portfolio at a specific location. it'll then create a new saved file.

Search Bar:

Allow the User to search for the any company with the name or Ticker symbol. After they choosen the Ticker. The graph will display the history of the Stocks price for Ticker symbol

Bottom Section:

Buy Shares:

Allow the User to Buy shares from the Ticker Symbol they have choosen that is display in the graph. Only allow the User to Buy more than 10 Shares, Otherwise the transaction won't be completed. If the User doesn't have enough balance to buy, the transaction will also won't be completed.

Sell Shares:

Allow the User to Sell shares from the Ticker Symbol they have choosen that is display in the graph. If the User doesn't have any Ticker Symbol Shares, the transaction won't be completed.

Enter Shares:

Provides the User to enter the amount of Shares in-order to Buy/Sell. Can't enter in a negative or 0 number, the transaction is allow if the User Portfolio have the balance.

Broker's' Fee:

An Option for the User if they want to turn on Broker's Fee. For Buy: Broker's Fee is 1% of the amount total cost fee from Shares For Sell: Broker's Fee is $10 from the total amount of Selling Shares

Change Name:

Allow the User to change their Portfolio Name. Then allow the User to update Portfolio name file, if not, the Portfolio name is changed.

Help:

About:

This provides information about the explanation overview of the Program, updates, Versioning, and Author for the program.

How to Use:

This provides information on it's feature & how to play the Stock Markete Simulator.

GUI Properties:

Graph:

The middle Graph provide for the User to display the details of daily open/high/low/close/volume values, daily adjusted close values, and historical split/dividend events of the global equity specified from the User choosen Ticker Symbol.

Ticker Search

The Ticker Search feature is for the User to search for a specific Company by Name or Ticker to allow the User to Choose the company in order to be display in the graph displaying the Stock's history. Allow the User to make the assumption in order to Buy/Sell.
